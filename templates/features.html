AI Report Generator

Automated Performance Test Report Generation

  • Generate comprehensive performance test reports
  • AI-powered analysis of response times, throughput, and error rates
  • Automatic compliance checking against SLA thresholds
  • Detailed recommendations for performance optimization
  • Exportable reports in multiple formats (PDF, HTML, DOCX)
How to Generate a Report:
  1. Upload your JMeter HTML report as .zip file
  2. add fields that relates to this test round
  3. You have the option to check if you need GPT-4o analysis
  4. Click "Generate Report" and wait for for the page to load
  5. Download will start automatically for the generated report

XML Correlation Finder

How to Find Correlations:
  1. Upload your XML file generated by JMeter Recorder
  2. Add URLs filters if needed comma separated
  3. You can add a mutual substring in all URLs you need to include
  4. Run the correlation analysis
  5. Review the detected correlations
